HTML Lists

  • <ol> - ordered list
  • <ul> - unordered list
  • <dl> - description list
  • <li> - list item
  • <dt> - description item
  • <dd> - description data

HTML Tables

  • table: < table >

  • table cell: < td >
  • table footer: < tfoot >
